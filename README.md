# Snake Game
A browser-based Snake game built using HTML, CSS, and JavaScript, where the player controls the snake to collect items and increase their score while avoiding collisions.

## Project Overview
This project recreates the classic Snake game using JavaScript logic to control movement, collision detection, and score progression. The game updates dynamically based on user input and game state.

## Technologies Used

- CSS
- JavaScript
